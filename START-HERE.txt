🚀 立即开始 (3个步骤，10分钟)

第 1 步：安装 PostgreSQL (5 分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  
  Windows:
    → 访问 https://www.postgresql.org/download/windows/
    → 下载并运行安装程序
    → 记住 postgres 用户密码
  
  macOS:
    → brew install postgresql@15
    → brew services start postgresql@15
  
  Linux:
    → sudo apt-get install postgresql
    → sudo service postgresql start


第 2 步：初始化数据库 (2 分钟)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  
  打开 PowerShell 或命令行，进入 ProofRead-Backend 目录
  
  推荐方式 (Windows):
    → .\init-db.ps1
  
  或使用 Python (跨平台):
    → pip install python-dotenv psycopg2-binary
    → python setup-db.py
  
  或使用批处理 (Windows 备选):
    → init-db.bat
  
  脚本会自动:
    ✓ 创建数据库用户 (proofread_user)
    ✓ 创建数据库 (proofread_db)
    ✓ 创建 5 个表和索引
    ✓ 配置权限


第 3 步：启动后端服务 (1 分钟)

  npm install  # 如果未安装
  npm run dev
  
  应该看到:
    ✓ PostgreSQL 连接成功

    ✓ http://localhost:3001

🗃️ 数据库架构
═══════════════════════════════════════════════════════════════════════════

5 个核心表:

  users (用户账户)
  ├─ id (自增主键)
  ├─ account (账号)
  ├─ password_hash (加密密码)
  ├─ token_balance (Token 余额)
  └─ 索引: idx_users_account

  token_logs (Token 消费日志)
  ├─ user_id (外键)
  ├─ deduction_amount (扣款金额)
  ├─ remaining_balance (剩余余额)
  └─ 索引: idx_token_logs_user_id, created_at

  payment_orders (充值订单)
  ├─ order_number (订单号)
  ├─ user_id (外键)
  ├─ order_status (订单状态)
  ├─ expired_at (过期时间)
  └─ 索引: idx_payment_orders_number, status

  recharge_logs (充值记录)
  ├─ user_id (外键)
  ├─ recharge_amount (充值金额)
  ├─ payment_status (支付状态)
  └─ 索引: idx_recharge_logs_user_id

  login_logs (登录日志)
  ├─ user_id (外键)
  ├─ login_ip (登录 IP)
  ├─ login_time (登录时间)
  └─ 索引: idx_login_logs_user_id, login_time

🔍 常见任务
═══════════════════════════════════════════════════════════════════════════

查看数据库中的用户:
  psql -h localhost -U proofread_user -d proofread_db \
    -c "SELECT id, account, token_balance FROM users;"

查看 Token 消费记录:
  psql -h localhost -U proofread_user -d proofread_db \
    -c "SELECT * FROM token_logs LIMIT 10;"

重置用户 Token 余额:
  psql -h localhost -U proofread_user -d proofread_db \
    -c "UPDATE users SET token_balance = 10000 WHERE account = 'test';"

重新初始化数据库:
  psql -h localhost -U postgres \
    -c "DROP DATABASE IF EXISTS proofread_db;"
  .\init-db.ps1


🛠️ 故障排除
═══════════════════════════════════════════════════════════════════════════

问题: psql 命令未找到
  → PostgreSQL 未安装或未添加到 PATH
  → 重新安装 PostgreSQL

问题: 无法连接到数据库
  → PostgreSQL 服务未运行
  → Windows: 检查服务管理器
  → Mac: brew services restart postgresql@15
  → Linux: sudo service postgresql restart

问题: 初始化脚本失败
  → 检查 postgres 用户密码是否正确
  → 查看 POSTGRESQL-SETUP.md 故障排除部分
  → 尝试另一个初始化脚本

问题: 后端启动失败
  → 确认数据库初始化成功
  → 检查 .env 配置是否正确
  → 查看终端错误日志
